企业报表网站 库存、订单、营业额报表汇总

一,系统简介 二,技术特点 三,系统画面

一,系统简介

1，包含各类统计报表，数量60+

2，报表中包含统计图表和数据表格

3，报表权限分部门，分人员进行权限管控

二,技术特点

1，类型丰富,包含不同类型的图表，更直观的显示数据

2，权限管控，权限分层管理，保证数据安全

3，数据缓存，对统计耗时较长的报表，缓存报表数据

★ 开发工具及使用技术

 
开发工具：Visual Studio 2010、PL SQL
数据库：Oracle、SQL Server  
开发语言：C#、ASP.NET
图表工具：Microsoft Chart
