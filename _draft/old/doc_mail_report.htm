邮件报表系统 及时推送、分层管理、图表结合

一,系统简介 二,技术特点 三,系统画面

一,系统简介

1，系统中包含报表有50+

2，即时推送，系统会在第一时间即时把各个部门的报表发送到对应收件人

3，分层管控，对于每个报表的收件人有明确管控，并对某些报表分版本来发送

4，图表结合，邮件内容不仅包含数据表信息，还包含图表信息，更能直观的显示数据的变化趋势

二,技术特点

1，邮件报表中数据与图表的汇整发送

2，报表分时间段，分频率的定时发送

3，报表发送后的自动检查及即时补发

4，记录报表发送日志及部分报表内容日志

★ 开发工具及使用技术

 
开发工具：Visual Studio 2010、PL SQL
数据库：Oracle、SQL Server
开发语言：C#
图表工具：Microsoft Chart
